# YouTube OAuth Verify

This repository hosts static verification and policy pages used for YouTube OAuth app setup.

## Purpose

- Publish a public privacy policy page for OAuth review
- Serve domain verification files required by Google (and related platforms)

## Files

- `index.html` - privacy policy page
- `google-site-verification.html` - Google Search Console / domain verification file
- `tiktokOyyEujRc2nri1YzxIeWoCeN5c9oJGS7I.txt` - third-party verification token file
- `.nojekyll` - ensures GitHub Pages serves files as-is

## Deployment

Designed for GitHub Pages static hosting.
